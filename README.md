# Auto Repair Invoice

## Prerequisites

https://tauri.studio/en/docs/getting-started/setup-windows

1. WebView2
2. Rustup
3. Microsoft Visual Studio C++ build tools
4. npm OR yarn

## Steps  
`yarn install`  
`yarn tauri info`  

## Local

Run the following commands in two separate terminals

`yarn tauri dev`
